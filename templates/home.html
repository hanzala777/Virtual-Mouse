Virtual Mouse

This Project is about, implementing work of mouse using hand gestures.

Guide

Virtual Mouse Project

Using Hand Gesture you can operate mouse operation like
1.LeftClick
2.RightClick
3.Drag&Drop
4.ScrollUp & ScrollDown

After Clicking Start Button Your Camera Will Open

Manual Guide

Gesture 1

Using This hand gesture, You can move cursor of the computer screen
If you move hand then relatively the cursor wil move
Note that if keep index finger closer to middle finger then curser will not move or stop at that position.

Gesture 2

Using This Gesture You can right-click on the screen respective of cursor position

Gesture 3

Using This Gesture You can left-click on the screen respective of cursor position

Gesture 4

Using This gesture into the camera you can scroll up the screen

Gesture 5

Using gesture into the camera you can scroll down the screen

Gesture 6

Using gesture when cursor is on any item then you can drag that item and move hand respective of screen and if you to drop the item just apply gesture 1

Note:To turn off the program click on the camera frame then press 'Q'
